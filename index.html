MetaSync

MetaSync is a secure and reliable file synchronization service that uses multiple cloud synchronization services as untrusted storage providers.

View the Project on GitHub UWNetworksLab/metasync

Instruction

Warning! MetaSync is currently alpha version! Use it only for research-purpose, and do not put any important files under MetaSync repository.

Prerequisite

Publication

  • Seungyeop Han, Haichen Shen, Taesoo Kim, Arvind Krishnamurthy, Thomas Anderson, and David Wetherall. MetaSync: File Synchronization Across Multiple Untrusted Storage Services. Technical Report, UW-CSE-14-05-02, May 2014
